HTML

HTML viết tắt của Hypertext Markup Language là ngôn ngữ lập trình dùng để xây dựng và cấu trúc lại các thành phần có trong Website.

CSS

CSS là từ viết tắt của Cascading Style Sheets là ngôn ngữ được sử dụng để tạo nên phong cách cho website.

Javascript

JavaScript là một ngôn ngữ lập trình website, được tích hợp và nhúng trong HTML giúp website sống động hơn.
